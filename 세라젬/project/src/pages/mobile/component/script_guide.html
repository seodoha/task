바텀시트 팝업

열기 : javascript function을 이용한 예

uiCommon.layer.open을 호출시에 매개변수로 열고 싶은 레이어팝업의 아이디명을 입력하면 된다.
ex) <a href="javascript: uiCommon.layer.open('test-layer');">javascript function open</a>

Layer Popup open function open

닫기 : Jquery click 이벤트를 이용한 예

클래스명 'popup-close'를 붙여주면 해당 레이어 팝업은 닫힌다.
ex) <a href="javascript:;" class="popup-close">jquery click close</a>

닫기 : javascript function을 이용한 예

uiCommon.layer.close 호출 할때 매개변수로 타겟레이어의 클래스를 넣어준다
ex) <a href="javascript: uiCommon.layer.close('layer');">javascript function close</a>

토스트 팝업 가이드

열기 : data-toast-pop의 속성값에 열고 싶은 토스트팝업의 아이디명을 입력한다.
ex) <a href="javascript:;" data-toast-pop="test-toast">jquery click open</a>

열기 : uiCommon.toast.open을 호출시에 매개변수로 열고 싶은 토스트팝업의 아이디명을 입력하면 된다.
ex) <a href="javascript: uiCommon.toast.open('test-toast');">javascript function open</a>

슬릭 호출 예
