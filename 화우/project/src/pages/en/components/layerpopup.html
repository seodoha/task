GUIDE

열기

uiCommon.layer.open을 호출시에 매개변수로 열고 싶은 레이어팝업의 아이디명을 입력하면 된다.
ex) <a href="javascript: uiCommon.layer.open('test-layer');">javascript function open</a>


uiCommon.layer.open을 호출시에 매개변수로 열고 싶은 레이어팝업의 아이디명을 입력하면 된다.
ex) <a href="javascript: uiCommon.layer.open('test-layer');">javascript function open</a>

닫기

클래스명 'popup-close'를 붙여주면 해당 레이어 팝업은 닫힌다.
ex) <a href="javascript:;" class="popup-close">jquery click close</a>


uiCommon.layer.close 호출 할때 매개변수로 타겟레이어의 클래스를 넣어준다
ex) <a href="javascript: uiCommon.layer.close('layer');">javascript function close</a>

웹접근성

법무법인(유) 화우는 장애인 및 비장애인이 동등하게 웹서비스를 이용하실 수 있도록, 웹접근성 지침을 준수하여 웹사이트를 제작하였습니다. 앞으로도 고객의 편리한 이용을 위해 지속적으로 개선해 나갈 예정입니다.

최종제출 후 지원서를 수정하실 수 없습니다. 최종제출 전 미리보기로 출력할 수 있습니다. 작성하신 지원서를 최종제출 하시겠습니까?
