Slick

GUIDE

슬라이더 플러그인은 Slick을 사용하며,
마크업 구조는 slider-wrap 안에 지정할 슬릭 target을 놓는 구조이다.
열기 : uiCommon.slick.init 함수를 호출하면서 첫번째 매개변수로 target의 jquery 객체와 두번째 매개변수로 slick 옵션을 객체로 넘기면 된다.
ex) uiCommon.slick.init($('.slider-list'), { infinite: false, speed: 500, slidesToShow: 4, dots: false, });
